# Политика конфиденциальности приложения «Табель»

Дата вступления в силу: 15 июля 2026 г.

## 1. Общие положения

Приложение «Табель» (далее — Приложение) предназначено для персонального
ведения учёта личного состава подразделения и работает полностью офлайн.

## 2. Какие данные обрабатывает Приложение

Пользователь самостоятельно вносит в Приложение данные: фамилии, имена,
отчества, воинские звания, должности, сведения об отпусках, командировках,
наградах, закреплённом вооружении и параметрах денежного довольствия.

**Все данные хранятся только в памяти устройства пользователя** в локальной
базе данных Приложения.

## 3. Что Приложение НЕ делает

- НЕ передаёт данные в интернет: у Приложения нет сетевых функций и нет
  разрешения на доступ в сеть для обмена данными.
- НЕ собирает аналитику, телеметрию и статистику использования.
- НЕ содержит рекламы и сторонних SDK-трекеров.
- НЕ запрашивает доступ к контактам, геолокации, камере, микрофону.
- НЕ создаёт учётных записей и не требует регистрации.

## 4. Резервные копии

Экспорт данных (файлы `.tbl`, PDF, Excel) выполняется только по явной команде
пользователя через системное меню «Поделиться». Куда передаётся такой файл —
решает пользователь. Файл `.tbl` может быть зашифрован паролем (AES-GCM);
без пароля его содержимое прочитать нельзя.

Системное резервное копирование Android для Приложения отключено
(`allowBackup=false`): данные не копируются в облако автоматически.

## 5. Защита данных

Доступ к Приложению можно ограничить цифровым кодом. Код хранится в виде
криптографической свёртки (PBKDF2), восстановить его нельзя. Снятие
блокировки без кода возможно только с полным удалением всех данных.
Снимки экрана Приложения заблокированы системным флагом FLAG_SECURE.

## 6. Удаление данных

Все данные удаляются средствами самого Приложения (Настройки → «Удалить всё»)
либо при удалении Приложения с устройства.

## 7. Ответственность пользователя

Пользователь самостоятельно отвечает за состав вносимых данных и за
соблюдение применимых к ним требований (в том числе режима защиты сведений,
если таковой установлен) на своём устройстве.
